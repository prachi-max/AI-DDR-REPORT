# AI DDR Report Generator

An AI-powered system that analyzes building inspection and thermal reports to generate a structured Detailed Diagnostic Report (DDR).

## 🚀 Features
- PDF text extraction
- AI-based issue detection
- Data merging and reasoning
- Professional report generation
- Modern UI dashboard

## 🧠 Tech Stack
- Python (Flask)
- Ollama (Local LLM - llama3 / mistral)
- HTML, CSS
